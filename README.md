## 📋 Requirements

To run the tap-ducky application smoothly, ensure your Android phone meets the following requirements:

- **Android Version**: Your device must run Android 5.0 (Lollipop) or higher.
- **Root Access**: The app requires root access for full functionality.
- **USB OTG Support**: Your device should support USB On-The-Go to use USB devices directly.
- **Storage Space**: Ensure you have at least 50 MB of free storage for installation.
